SteganVision Detailed Model Architectures

1. Encoder Architecture (Generator)

Type: U-Net with Skip Connections

Goal: Embed Secret Image into Cover Image while preserving details.

Input: Cover (3ch) + Secret (1ch) = 4 channels.

graph TD
    subgraph Input
    Cover[Cover Image (3ch)]
    Secret[Secret Image (1ch)]
    Concat[Concatenate (4ch)]
    end

    subgraph Encoder_UNet [U-Net Encoder]
    InitConv[Init Conv (64ch)]
    
    subgraph Downsampling
    D1[DownBlock 1 (128ch)]
    D2[DownBlock 2 (256ch)]
    D3[DownBlock 3 (512ch)]
    end
    
    subgraph Bottleneck
    B1[ConvBlock (512ch)]
    B2[ConvBlock (512ch)]
    end
    
    subgraph Upsampling
    U1[UpBlock 1 (256ch)]
    U2[UpBlock 2 (128ch)]
    U3[UpBlock 3 (64ch)]
    end
    
    OutputConv[Final Conv + Tanh]
    end

    Cover --> Concat
    Secret --> Concat
    Concat --> InitConv
    InitConv --> D1
    D1 --> D2
    D2 --> D3
    D3 --> B1
    B1 --> B2
    B2 --> U1
    U1 --> U2
    U2 --> U3
    U3 --> OutputConv
    
    OutputConv --> Stego[Stego Image (3ch)]

    %% Skip Connections
    D3 -.-> U1
    D2 -.-> U2
    D1 -.-> U3

2. Decoder Architecture

Type: Deep CNN

Goal: Extract Secret Image from Stego Image.

Input: Stego Image (3 channels).

    L3[Conv (128ch)]
    L4[Conv (128ch)]
    L5[Conv (256ch)]
    L6[Conv (256ch)]
    L7[Conv (128ch)]
    L8[Conv (64ch)]
    Out[Conv 1x1 + Tanh]
    end
    
    subgraph Output
    RecSecret[Reconstructed Secret (1ch)]
    end
    
    Stego --> L1
    L1 --> L2
    L2 --> L3
    L3 --> L4
    L4 --> L5
    L5 --> L6
    L6 --> L7
    L7 --> L8
    L8 --> Out
    Out --> RecSecret

3. Discriminator Architecture (GAN)

Type: PatchGAN with Spectral Normalization

Goal: Distinguish between Real Cover and Fake Stego.

Output: 16x16 Patch Predictions (averaged to single score).

graph TD
    subgraph Input
    Img[Input Image (3ch)]
    end
    
    subgraph Discriminator_PatchGAN [PatchGAN Discriminator]
    C1["Conv 4x4, Stride 2 (64ch)"]
    C2["Conv 4x4, Stride 2 (128ch)"]
    C3["Conv 4x4, Stride 2 (256ch)"]
    C4["Conv 4x4, Stride 2 (512ch)"]
    OutConv["Conv 3x3, Stride 1 (1ch)"]
    Avg[Global Average Pooling]
    Sigmoid[Sigmoid Activation]
    end
    
    subgraph Output
    Score[Real/Fake Score]
    end
    
    Img --> C1
    C1 --> C2
    C2 --> C3
    C3 --> C4
    C4 --> OutConv
    OutConv --> Avg
    Avg --> Sigmoid
    Sigmoid --> Score
